# Documentation & Logbook Difficulty Matrix

## Purpose

This repository studies how difficult it is for professionals to maintain
operational records while simultaneously performing their primary duties.

The focus is not merely the job itself, but the difficulty of maintaining
accurate documentation, attendance sheets, journals, registers, logbooks,
case sheets, trip reports, and operational records under real-world conditions.

Documentation Difficulty & Logbook Matrix

Overview

This repository ranks professions by the difficulty of maintaining documentation while simultaneously performing operational duties.

The ranking focuses on:

- Cognitive workload
- Physical workload
- Safety-critical decision making
- Environmental hazards
- Documentation opportunity
- Automation requirements

Conclusion

As cognitive workload increases, responsibility for documentation shifts away from the operator and toward automation, telemetry systems, event recorders, voice capture, observers, and AI-assisted logging systems.

The most demanding professions require machine-generated records because manual documentation would compromise operational safety, mission success, or human life.
